Perguntas Frequentes

Para se registrar como freelancer, siga os seguintes passos:

  1. Clique no botão "Registrar-se".
  2. Você será redirecionado para a página de registro.

Nesta página, insira seus dados pessoais nos campos apropriados. Isso inclui informações como:

  • Nome
  • Endereço de e-mail
  • Senha
  • E qualquer outra informação necessária.

Se preferir, em vez de clicar no botão "Registrar-se", você também pode acessar diretamente a página de registro através do seguinte link:

Clique aqui para acessar a página de cadastro de talento

Para se registrar como contratante, siga os seguintes passos:

  1. Clique no botão "Registrar-se".
  2. Você será redirecionado para a página de registro.

Nesta página, insira seus dados pessoais nos campos apropriados. Isso inclui informações como:

  • Nome
  • Endereço de e-mail
  • CNPJ
  • Senha
  • E qualquer outra informação necessária.

Se preferir, em vez de clicar no botão "Registrar-se", você também pode acessar diretamente a página de registro através do seguinte link:

Clique aqui para acessar a página de cadastro de empresa

Para atualizar seu perfil como freelancer, siga estes passos:

  1. Localize o botão identificado como "Atualizar Dados" e clique nele.
  2. Isso o levará à página de atualização de informações.

Na página de atualização, insira os dados que deseja modificar nos campos correspondentes. Certifique-se de fornecer as informações corretas e atualizadas.

Alternativamente, se preferir, você pode acessar diretamente a página de atualização por meio do seguinte link:

Clique aqui para acessar a página de atualização

Para cadastrar seu serviço, siga estes passos:

  1. Clique no botão designado para cadastrar o serviço.
  2. Isso o direcionará para a página de cadastro de habilidades.

Na página de cadastro, forneça os detalhes do seu currículo, incluindo suas habilidades e informações relevantes.

Se preferir, você também pode acessar diretamente a página de cadastro de habilidades por meio do seguinte link:

Clique aqui para acessar a página de cadastro de habilidades

Certifique-se de inserir as informações de forma completa e precisa para que seu currículo seja registrado corretamente.

Para encontrar jovens talentos em busca de emprego ou empresas que estão procurando serviços, siga estas etapas simples:

  1. Escolha uma das opções abaixo, dependendo do que você está procurando:
    • Se deseja encontrar jovens talentos, clique aqui.
    • Se está em busca de serviços oferecidos por empresas, clique aqui.
  2. Explore os perfis ou serviços disponíveis para encontrar a melhor correspondência para suas necessidades.

Essas etapas simples tornarão mais fácil encontrar exatamente o que você procura.
